# WealthCalc サポートページ

WealthCalc は、資産運用の将来推移を簡単にシミュレーションできる iOS アプリです。

## 主な機能
- 初期投資額、年利、毎月の積立による資産推移を計算
- グラフによる視覚的な表示
- Pro機能として、追加の積立シミュレーションに対応

## よくある質問
**Q. データはどこかに送信されますか？**  
A. いいえ。すべての計算は端末内で行われ、外部に送信されることはありません。

**Q. Pro機能は何ですか？**  
A. 2つ目の積立条件を追加できる機能です。
